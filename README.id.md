# Awesome Sentrix

> Daftar terkurasi resmi untuk **Sentrix Chain** — sumber daya, alat developer, infrastruktur, aplikasi, dan panduan.

Halaman ini adalah ringkasan Bahasa Indonesia. Daftar lengkap ada di [README.md](README.md).

**Sentrix Chain** adalah Layer-1 berbasis Rust, kompatibel EVM, dengan fokus pada protocol engineering, infrastruktur validator, kompatibilitas RPC, staking, dan tooling developer.

## Untuk siapa?

| Kamu seorang… | Mulai dari |

| **Developer Solidity** membangun dApp di Sentrix | [Developer Tools](README.md#developer-tools) → [Tutorials](README.md#tutorials) → [dApp Starter](https://github.com/SentrisCloud/dapp-starter) |
| **Developer Rust** integrasi dengan node Sentrix | [Rust SDK](https://github.com/SentrisCloud/sdk-rs) → [gRPC endpoints](README.md#networks) |
| **Operator node** menjalankan validator atau fullnode | [Running a Node](README.md#running-a-node) |
