# Contributing to Awesome Sentrix

Thanks for helping improve Awesome Sentrix.

This repository is a curated list, not a complete directory of every Sentrix-related link. Please only add resources that are useful, maintained, and relevant for Sentrix Chain developers, validators, users, or ecosystem contributors.

## What can be added?

Good additions include:

- official Sentrix resources
- developer tools
- tutorials
- SDK examples
- validator guides
- RPC examples
- ecosystem applications
- infrastructure tools
- wallet integration notes
- security and audit resources

## Quality bar

Before submitting a resource, make sure it is:

- directly related to Sentrix Chain
- publicly accessible
- useful to developers, validators, users, or ecosystem contributors
- not misleading
- not abandoned, unless clearly marked as archived or historical
- not purely promotional
